# Pasta de Texturas

Coloque sua textura MatCap aqui.

## Arquivo Esperado

- `clay-white.jpg` — Textura MatCap para efeito "argila branca"

## Requisitos da Textura

- **Formato:** JPEG (preferido por tamanho) ou PNG
- **Resolução:** 512x512 pixels (não precisa mais que isso)
- **Tipo:** Textura MatCap (esférica, iluminação embutida)
- **Estilo:** Cerâmica fosca / Clay / Primer automotivo

## Onde encontrar MatCaps

- https://github.com/nidorx/matcaps
- Criar no Blender (render de esfera com seu material)

## Recomendação

Procure por "Ceramic White MatCap" ou "Clay MatCap" para o estilo desejado.
